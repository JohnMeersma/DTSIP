# DTSIP Ansible Playbooks

This repository contains Ansible playbooks to configure multiple Cisco 1000v routers. Follow the steps below to set up your environment and run the playbooks.

## Prerequisites

- AWS Linux instance
- Git
- Python 3
- Ansible
